# 교내 코드 저장 아카이브 및 AI 기반 검색 시스템

![Project Banner](banner.png)

---

## 🔹 프로젝트 개요
본 시스템은 교내 학생 프로젝트의 코드와 산출물을 체계적으로 아카이브하고, AI 기반 검색 및 추천 기능을 제공하여 학습과 연구를 지원합니다.  
프로젝트 정보, 코드, 산출물, 기술 스택 등 다양한 데이터를 효율적으로 저장하고, 자연어 질의 기반 검색 및 개인화 추천을 통해 학습 환경을 개선합니다.

---

## 📚 주요 기능

### 1. 코드/산출물 저장 및 AI 검색
- **데이터 저장 구조**
  - 프로젝트 메타데이터: PostgreSQL
  - 코드/파일: 객체 스토리지 (MinIO)
- **검색 기능**
  - Elasticsearch/OpenSearch 기반 키워드 검색
  - `bge-small`, `e5-small` 임베딩 모델을 활용한 벡터화
  - pgvector 또는 Qdrant를 활용한 유사도 기반 검색
  - 기술 스택, 과목, 연도 등 필터 적용 가능
- **RAG 기반 대화형 검색**
  - Claude AI로 사용자 의도 및 조건 추출
  - 벡터 검색 + 메타데이터 검색 → 관련 프로젝트 후보 반환
  - LLM 기반 요약 및 후속 질의 맥락 유지

### 2. 사용자 맞춤 프로젝트 추천 및 학습 지원
- 개인화된 탐색 환경 제공
- 검색 패턴, 관심 기술 스택, 열람 기록 기반 추천
- 유사 기술 스택/문제 해결 방식 기반 추천 알고리즘
- 프로젝트 구조 및 주요 코드 스니펫 LLM 자동 요약
- 학습 가이드 및 관련 튜토리얼 연결 가능

### 3. 대화형 인터페이스 기반 지식 탐색
- GPT 계열 모델 기반 자연어 질의 응답
- 프로젝트 설계서, README, 기술 문서에서 핵심 정보 추출
- 코드 구조 및 기술 개념 자동 해설
- 검색, 추천, 비교, 학습 가이드 등 맞춤 대화 플로우 제공
- 사용자 로그 분석을 통한 응답 품질 및 추천 정확도 개선

### 4. 지속적인 프로젝트 업로드 유도
- 학기별 프로젝트 반복 업로드 유도
- 게이미피케이션 요소: 배지, 포인트 시스템
- 학과/팀 단위 랭킹으로 참여 동기 강화

## 🛠 기술 스택
- **Backend:** Python, FastAPI, PostgreSQL, MinIO  
- **Search & Embedding:** Elasticsearch/OpenSearch, pgvector, Qdrant, bge-small, e5-small  
- **AI/NLP:** Claude AI, GPT 계열 LLM  
- **Frontend:** React, Tailwind CSS (선택적)

## ⚡ 사용 예시
1. **검색**
   - "Python 기반 웹 크롤러 프로젝트"
   - 벡터 + 메타데이터 기반 후보 프로젝트 반환
2. **추천**
   - 관심 기술 스택: `React + FastAPI`
   - 유사 프로젝트 자동 추천 및 학습 가이드 제공
3. **대화형 질의**
   - "이 프로젝트에서 사용하는 주요 알고리즘이 뭐야?"
   - LLM이 코드 구조와 알고리즘 설명 제공
